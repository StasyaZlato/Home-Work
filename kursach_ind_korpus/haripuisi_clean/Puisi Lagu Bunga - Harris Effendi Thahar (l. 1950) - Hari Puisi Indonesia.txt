Author: Harris Effendi Thahar

Name: Lagu Bunga

Sumber: Lagu Sederhana Merdeka (Penerbitan keduapuluh lima Puisi Indonesia, dibantu oleh Bidang Kesenian Kanwil Dep. P &amp; K Sumatera Barat, dan Pusat Kesenian Padang; 1979)

Year: 1975.

URL: http://www.haripuisi.com/arsip/3812

Poem text: 
Lagu bunga sebelum layu
Lagu duka sesudah itu
Lagu sangsi nyalakan api
Tentang kupu dan bunga
Tentang kehilangan semata
Tentang dendamku yang tak mempan
pada wewangian
Bungaku kesangsian
Kupuku api itu
Lagu bunga sebelum layu
Lagu duka sesudah itu
Lagu sangsi nyalakan api
Lagu bunga sebelum layu
Lagu duka sesudah itu
Lagu sangsi nyalakan api
Tentang sebuah kehidupan.
